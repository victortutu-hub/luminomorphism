l-neural-progress

Progress bar ca rețea neurală cu propagare de impulsuri și învățare dinamică. Simulează procesarea informației printr-o rețea de neuroni interconectați.

Configurații și Stări

🔄 Loading State

Rețea în curs de procesare cu pulsuri automate și noduri care se activează progresiv.

✅ Success State

Procesare completă cu succes - toate nodurile active și efecte de celebrare.

❌ Error State

Stare de eroare cu animații specifice și culori de alertă.

🧠 Funcționalități Rețea Neurală
  • Propagare Progresivă: Activarea se propagă de la input la output
  • Pulsuri Dinamice: Impulsuri vizuale prin conexiuni active
  • Learning Mode: Conexiunile "învață" și se întăresc în timp
  • Stări Multiple: Loading, Success, Error cu animații specifice
  • Auto-Animate: Generare automată de pulsuri
📝 Utilizare în Cod
<l-neural-progress 
    value="45"
    max="100"
    state="loading"
    nodes="8"
    layers="4"
    speed="1.5"
    color="#00ffff"
    pulse-intensity="1.2"
    auto-animate="true"
    learning-mode="true">
</l-neural-progress>

<script>
const progress = document.querySelector('l-neural-progress');

// API Methods
progress.setValue(75);
progress.setState('success');
progress.pulse();    // Generează val de pulsuri
progress.learn();    // Activează learning mode temporar
progress.reset();    // Reset la 0

// Event Listeners
progress.addEventListener('progress-change', e => {
    console.log('Progress:', e.detail.percentage + '%');
});

progress.addEventListener('progress-complete', e => {
    console.log('Procesare completă!');
});
</script>
⚙️ Atribute Disponibile
value - Valoarea curentă (0-max)
max - Valoarea maximă
state - loading|success|error|idle
nodes - Numărul de noduri per layer
layers - Numărul de layer-uri
speed - Viteza animațiilor
color - Culoarea efectelor
pulse-intensity - Intensitatea pulsurilor
auto-animate - Pulsuri automate
learning-mode - Conexiuni care învață
