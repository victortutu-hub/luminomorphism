üî¨ Quantum Mechanics Visualization

This component visualizes several quantum mechanical concepts:

  • Superposition: The toggle exists in multiple states simultaneously during transition
  • Wave Function Collapse: The probability wave collapses into a definite state
  • Quantum Field: The surrounding energy field that influences particle behavior
  • Particle Emission: Virtual particles created during state transitions
‚öõÔ∏è Toggle Features

  • Wave Function Collapse: Animated wave that sweeps across the toggle

  • Quantum Field: Pulsating energy field around the toggle
  • Configurable Effects: Adjustable particle count, speed, and intensity
  • Accessibility: Keyboard support and focus states
üìù Usage in Code
<l-quantum-toggle 
    checked
    size="large"
    color-on="#00ff80"
    color-off="#ff4444"
    quantum-mode="true"
    superposition-enabled="true"
    particle-count="12"
    transition-speed="1.5"
    glow-intensity="1.2"
    label="Quantum State">
</l-quantum-toggle>

<script>
const toggle = document.querySelector('l-quantum-toggle');

// API Methods
toggle.check();     // Set to checked
toggle.uncheck();   // Set to unchecked
toggle.toggle();    // Toggle state
toggle.isChecked(); // Get current state
toggle.disable();   // Disable toggle
toggle.enable();    // Enable toggle

// Event Listeners
toggle.addEventListener('change', e => {
    console.log('State:', e.detail.checked);
});
</script>
